## Backend Routes

### Home Route

http://localhost:3000/

Displays backend running message.

### About Route

http://localhost:3000/about

Displays student details.

### Health Route

http://localhost:3000/health

Returns server status.

## How to Run Project

### Backend Setup

cd backend  
npm install  
npm start

Server runs on:
http://localhost:3000
